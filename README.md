# CodeJam ⚔️

> **Stop watching tutorials. Start shipping logic under pressure.**


An open-source tool that makes the world of developers better!
(Education theme)

CodeJam is a gamified coding arena where learning feels like gameplay, debugging feels like combat, and progress is earned through action—not passive scrolling.

---

## 🎮 Why CodeJam?

Most beginners consume hours of tutorials and still freeze in front of a blank editor.

CodeJam flips that model:

- Learn by doing, not watching
- Build speed, logic, and confidence through real challenges
- Turn solo coding into a social, competitive loop

---

## 🧭 Choose Your Path

### 👨‍💻 I’m here to **play & improve**
- Start with roadmap skirmishes
- Unlock higher tiers by proving skills
- Enter boss battles to solve real incident-style problems

### 🧠 I’m here to **learn interactively**
- Use AI-powered guides for concept-by-concept progression
- Get dynamic challenge variations to avoid repetitive practice

### 🤝 I’m here to **compete with friends**
- Challenge friends asynchronously in Ghost Mode
- Trigger revenge loops and climb social rankings

---

## 🚀 Core Game Modes

### 1) Roadmap Skirmishes
A structured journey where each level unlock requires solving practical coding tasks.

### 2) Boss Battles (Daytona-powered)
Fight live debugging incidents in secure ephemeral Linux sandboxes.

### 3) Ghost Mode (Async Multiplayer)
Set a score now, challenge your friend later, and keep the rivalry alive through revenge rounds.

### 4) Social Engine
Presence, friend system, and challenge notifications that keep engagement continuous.

## 🏗️ Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS, Framer Motion
- **Backend & Realtime:** Convex
- **Sandbox Infra:** Daytona SDK
- **AI Layer:** OpenRouter + Google GenAI integrations

---

## 🔍 How CodeJam Works

1. Pick a learning track from the roadmap.
2. Solve interactive coding skirmishes.
3. Unlock advanced tiers and boss incidents.
4. Challenge friends in async ghost battles.
5. Repeat with increasing difficulty and better speed.
